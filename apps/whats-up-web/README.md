# whats-up-web

This is the main web app for whats-up-xyz, built with Next.js.

View the production app at [whats-up.xyz](https://whats-up.xyz)

See Deployment details at https://vercel.com/bren96s-projects/whats-up-xyz.


## Unit Testing

This app uses [Jest](https://jestjs.io/) for unit testing.

To run the unit tests:
```
nx test whats-up-web
```

Followed implementation example on [Primsa Client Unit Testing](https://www.prisma.io/docs/orm/prisma-client/testing/unit-testing#example-unit-tests).
